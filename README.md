# Unity3D Terrain Practice
This project was made for me to practice unity 3D's terrain function and a few other packages. This project focused on a few features:
- Creating a body of water
- Terrain feature
  - Depth of Terrain
  - Flowers
  - Grass
  - Trees
- Adding wind-zone
- Using Skybox
